[=] Working Platforms

Windows
Linux
Termux

[+] Installation

1 > Go To The Inside Of The Folder Using 'cd' command
2 > pip install -r requirements.txt
3 > python bigman.py (For Windows)
4 > python3 bigman.py (For Linux & Termux)
5 > Enjoy!
